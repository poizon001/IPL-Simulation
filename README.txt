IPL Simultion
==============

Resources used
--------------
players.txt 
"player_id,player_name,player_type,player_price"

team.txt 
"TeamName"
"player_id"

Commands
--------
To compile source files
	javac *.java

To run Application
	java -cp . Simulation

Application Instructions
------------------------

To Add:
Click on checkboxes for selecting players and click on add button

To Remove:
Click on checkboxes for selecting players and click on remove button

To Clear:
Click on Clear for removing all entries

To set TeamName:
Enter TeamName in textfield and click submit
